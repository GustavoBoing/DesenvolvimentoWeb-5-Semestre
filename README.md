# 🌐 Desenvolvimento Web — FACENS

## 📚 Sobre a disciplina
A disciplina de **Desenvolvimento Web** tem como objetivo capacitar os alunos no planejamento, desenvolvimento e publicação de uma aplicação web profissional, utilizando tecnologias e práticas amplamente adotadas pelo mercado.

Durante o semestre será desenvolvido um projeto completo, passando por todas as etapas do ciclo de vida de um sistema web — desde a concepção até o deploy em ambiente de produção.

---

## 🎯 Objetivo principal
Projetar, desenvolver e publicar um **site profissional**, aplicando conceitos modernos de desenvolvimento web, arquitetura de software e gestão de versões.

---

## 🧠 Competências desenvolvidas

- Estruturação de projetos web
- Versionamento de código com Git
- Publicação de aplicações (Deploy)
- Configuração de hospedagem
- Utilização de CMS
- Desenvolvimento com frameworks modernos
- Organização de código e boas práticas
- Integração com serviços e APIs

---

## 🛠 Tecnologias e conceitos abordados

### 🔹 CMS (Content Management System)
- WordPress  
Criação e gerenciamento de sites dinâmicos sem a necessidade de desenvolvimento do zero.

### 🔹 Versionamento de código
- Git  
Controle de versões, trabalho em equipe e organização do fluxo de desenvolvimento.

### 🔹 Deploy e Hospedagem
- Publicação de aplicações web
- Configuração de servidores
- Ambiente de produção

### 🔹 Frameworks Front-end
- React
- Vue.js

Desenvolvimento de interfaces modernas, componentizadas e reativas.

### 🔹 Frameworks Back-end
Introdução ao desenvolvimento server-side e à integração com o front-end.

### 🔹 Bancos de Dados
- Introdução aos bancos de dados **NoSQL**

---

## 🚀 Projeto da disciplina
Ao longo do semestre será desenvolvido um **site profissional completo**, aplicando:

- Planejamento de layout e estrutura
- Desenvolvimento responsivo
- Gerenciamento de conteúdo
- Otimização para performance
- Publicação online

---

## 📈 Resultados esperados
Ao final da disciplina, o aluno será capaz de:

- Desenvolver e publicar um projeto web real
- Trabalhar com tecnologias modernas de mercado
- Utilizar versionamento de forma profissional
- Compreender a arquitetura básica de aplicações web
- Criar soluções escaláveis e organizadas
